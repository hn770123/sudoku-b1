# 数独ゲーム

9x9数独パズルゲーム - 自動生成で難易度表示付き

## 概要

バニラJavaScriptで実装されたブラウザベースの数独ゲームです。問題は自動生成され、難易度判定機能、PWA対応、レスポンシブデザインを備えています。

## 機能

### コア機能
- **パズル自動生成**: バックトラッキングアルゴリズムで完全なボードを生成し、40-60セルをランダムに間引いて問題を作成
- **難易度判定**: 間引いたセル数に基づいて難易度を判定
  - 簡単: 40-44セル
  - 普通: 45-51セル
  - 難しい: 52-60セル
- **解答チェック**: 入力された解答の正誤を即座に判定し、ビジュアルフィードバックを提供
  - 正解のセル: 緑色
  - 誤りのセル: 赤色
- **問題再生成**: ボタン一つで新しい問題を生成
- **進捗表示**: パズル生成中の進捗状況を表示

### PWA対応
- **インストール可能**: ブラウザからアプリとしてインストール可能
- **スタンドアロン動作**: インストール後はアプリとして起動
- **レスポンシブデザイン**: デスクトップ、タブレット、スマートフォンに対応

### UI/UX
- CSS Gridを使用した3x3ブロック境界の強調表示
- グラデーション背景とモダンなボタンデザイン
- モバイルデバイスでは自動的にボタンを縦配置に切り替え

## 使い方

### オンラインで使用
上記のデモURLにアクセスするだけで、すぐにプレイできます。

### ローカルで実行

1. リポジトリをクローン:
```bash
git clone https://github.com/hn770123/sudoku-b1.git
cd sudoku-b1
```

2. ローカルサーバーを起動:
```bash
# Pythonの場合
python -m http.server 8000

# Node.jsの場合
npx http-server

# その他のHTTPサーバーでも可
```

3. ブラウザで `http://localhost:8000` にアクセス

### 遊び方

1. ページを開くと自動的に数独問題が生成されます
2. 空いているマス（白いマス）に1-9の数字を入力
3. 「解答チェック」ボタンで正誤を確認
4. 「問題を再生成」ボタンで新しい問題に挑戦

## 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: CSS Grid、Flexbox、カスタムプロパティ
- **JavaScript (ES6+)**: バニラJavaScript、クラスベース設計
- **PWA**: Web App Manifest、Service Worker未使用（ブラウザキャッシュのみ）
